# Elite Skills Capture Widget

## Description
This widget automates the capture of elite skills from bosses throughout Guild Wars. It will automatically travel to the required maps, find the boss with the desired elite skill, and capture it using skill capture skills.

## Features
- **Automatic Elite Skill Detection**: Scans for all available elite skills that can be captured
- **Map Travel**: Automatically travels to the correct map for each elite skill
- **Boss Hunting**: Locates and approaches the boss that has the desired elite skill
- **Skill Capture**: Uses capture skills to learn the elite skill
- **Progress Tracking**: Keeps track of which skills have been captured
- **GUI Interface**: User-friendly interface for selecting and managing skill captures

## Usage
1. Open the Widget Manager in Py4GW
2. Navigate to Automation > Bots > Elite Skill Capture
3. Select the elite skill you want to capture from the list
4. Click "Start Capture" to begin the automation
5. The bot will handle travel, combat, and skill capture automatically

## Requirements
- Access to the required maps for elite skill capture
- Skill capture skills (Signet of Capture, etc.)
- Proper hero setup for combat support

## Notes
- The bot will skip skills that are already unlocked
- Map access requirements are checked before attempting capture
- The bot includes safety features and stuck detection
